# Proyecto Cloud - Despliegue en la Nube

## Descripción
Este proyecto demuestra el despliegue de una aplicación web con frontend y API REST en la nube.

## Tecnologías
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Despliegue**: Render (Plataforma en la nube)
